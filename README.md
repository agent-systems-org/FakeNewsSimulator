# FakeNewsSimulator

## Requirements
* Python 3.8
* Following packages
    * spade
    * numpy
    * networkx
    * matplotlib
    * flask
    * dash
    * plotly
    * sklearn

## How to run

First, start the visualization server:
```
python visualization/server.py -h
```

Then, start the simulation:
```
python main.py
```

Or:
```
python test_bots.py -h
```
